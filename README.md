# feedbackbot-reports

Auto-generated **public** GitHub Pages host for [FeedbackBot](https://github.com/eurekaffeine/FeedbackBot) reports.

- The FeedbackBot **pipeline source is private**. This repo holds only rendered, static HTML dashboards (daily review triage + backlog work queue).
- Reports are published from the OpenClaw host via `git push` (no GitHub Actions).
- Content is intentionally low-sensitivity: app-store feedback summaries, triage severity/priority, and links to (auth-gated) issues. **No LLM triage reasoning, no raw reviews, no PII.**
- Pages are served under an unguessable path and marked `noindex` / `robots: disallow`, so they are not listed here or indexed by search engines.

Do not edit by hand — files here are overwritten on each publish.
